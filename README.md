# Legal_Lens
Final Year Project . Legal Lens is an NLP application which gives sample verdict which judge will refer to fasten up the process.
# How to install
### clone git repository
> git clone https://github.com/Manojvbhat/Legal_Lens.git
### Open in editor
(VS code)
### cd to front_end
> cd front_end
### install dependencies
> node install
### run the dashboard using
> npm start
### cd back
> cd ..
### install dependencies for backend
>  npm install
### run Legal_lens.py (from Legal_Lens directory don't cd to server)
> python ./server/Legal_lens.py (or just press run button)
### cd to server
> node API_servers.js
